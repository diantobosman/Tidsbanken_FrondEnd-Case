## Assignment
Tidsbanken have approached the candidates and requested that a solution for managing
requests for vacation time for their employees. The application serves a dual purpose.
Firstly, it serves to manage the request and approval process of vacation days for employees and ease the burden of communication. Secondly, the application serves to inform employees about each otherâ€™s approved vacation days so that teams may adequately plan for periods of absence.
